# Writing Lambda functions
 
![[Pasted image 20251126130333.png]]
## Handler
- Like Java Main
- Limit what you include
- Function configuration
- Lambda-specific code
- No business logic
- Responsable de entender el entorno
## Controller
- Event Processing
- Core business logic
## Service
- External integrations
- Service abstractions
- Abstrae servicios externos para las funciones con las que interacciona mi codigo

## **Focus on services**
Every codebase is unique, but you will probably have multiple Lambda functions as well as other AWS resources per service. Give each service its own template, and use one repository per service and template.
